解决Springboot请求不到Controller

springboot最近很火，所以我也来学习一下springboot，同时也顺带感受一下idea的强大，说实话，在一个陌生的IDE里学习一个自己不熟悉的技术时，问题是真的MMP的多，从创建项目运行项目一直小问题不断，不过还好作为一个程序员有着基本的学习能力和解决问题能力，问题也就都解决了。最近就出现了在页面上请求不到controller的情况，出现404错误

如下图:
遂百度之，发现是启动类放的位置不对，导致controller没有被扫描到，把springboot的启动类放到controller包同级目录下，问题解决。
